# Examples

Use these prompts to sanity-check the command UX.

## Minimal

```text
/khon-party วิเคราะห์แผนนี้
```

## No-argument context mode

```text
/khon-party
```

## Deep debate

```text
/khon-party [depth=deep] เปรียบเทียบ 3 ทางเลือกนี้
```

## Forced personas

```text
/khon-party [personas=architect,operator,skeptic,governor] วิเคราะห์ระบบนี้
```

## Forced modules

```text
/khon-party [modules=cog.role_playing,cog.six_thinking_hats,cog.black_swan] วิเคราะห์โจทย์นี้
```
