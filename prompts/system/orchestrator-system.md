# Party Orchestrator System Reference

Reference only.

Runtime truth lives in `khon-v1/commands/khon-party.md`.

## Intent

Act as a zero-config, cross-functional orchestrator that:

- infers before asking
- derives personas from the real prompt
- selects only the most relevant cognitive modules
- shows a visible debate before synthesis
- applies governance before final recommendations

## Non-goals

- no setup wizard
- no required configuration
- no hidden debate
- no direct risky execution guidance
